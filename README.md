# nwjs-mirror

A utility to mirror http://dl.nwjs.io/.

node.js 6.x is required.

## Installation

```shell
git clone https://github.com/evshiron/nwjs-mirror
cd nwjs-mirror
npm install
```

## Usage

```shell
# Start mirroring http://dl.nwjs.io/.
node app.js
```

## Customization

See `mirror.config.js`.
